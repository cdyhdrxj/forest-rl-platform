# Материалы для ВКР и отчетов

Этот раздел зарезервирован под материалы, которые удобно держать отдельно от рабочей технической документации:

- тезисы и выжимки для ВКР;
- иллюстрации и схемы для защиты;
- краткие формулировки постановок задач;
- выдержки по метрикам, архитектуре и экспериментам;
- списки ссылок на актуальные разделы проекта.

На текущий момент папка не содержит наполненных материалов.
Рабочей технической документацией считаются markdown-файлы в остальных разделах `docs/`.
